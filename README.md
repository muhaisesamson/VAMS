# Uganda Veterans Affairs Management System

A front-end concept for a Uganda-themed veteran welfare portal.

## Structure
- `frontend/` — HTML, CSS, JavaScript UI
- `backend/` — backend separation placeholder and schema notes
- `frontend/assets/` — uploaded reference images used in the design

## Pages
- Home / landing page
- Register
- Login
- Dashboard
- Services
- Community
- Resources

## Notes
The prototype uses browser localStorage for demo-only login and registration flows.
